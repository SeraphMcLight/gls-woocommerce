== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gls-woocommerce` directory, or install the plugin through the WordPress plugins repository directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Woocommerce > Settings > GLS screen to configure the plugin
